# 業務タスク管理アプリ 発表

---

# アプリ概要

Laravel 13を用いて作成した、業務タスク管理アプリです。

ログインユーザーが自身の業務タスクを登録し、進捗状況や期限を管理できます。

また、管理者(admin)と一般ユーザー(user)でアクセス権限を分け、管理者は全ユーザーの状況確認、一般ユーザーは自身のタスク管理のみ行えるよう実装しました。

---

# デモ

## 一般ユーザー

- ログイン
- タスク一覧表示
- タスク追加
- タスク編集
- タスク削除
- タスク期限管理
- オンライン状態表示

## 管理者(admin)

- ダッシュボード表示
- ユーザー一覧表示
- オンライン/オフライン確認
- ユーザー検索
- ユーザーごとのタスク一覧確認

---

# 実装した機能

## 必須機能

- ユーザー登録
- ログイン/ログアウト
- タスクCRUD機能
- タスク一覧表示
- バリデーション
- 認証(auth)

## 追加機能

- admin/user 権限制御
- middlewareによる認可
- オンライン状態表示
- last_seen_at 管理
- ユーザー検索機能
- リアルタイム検索UI
- UUIDユーザー管理
- 管理者ダッシュボード
- ユーザー別タスク閲覧機能
- hover UI
- status circle UI
